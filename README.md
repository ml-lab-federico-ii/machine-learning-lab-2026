# Laboratorio di Machine Learning per l’Analisi Finanziaria — Università Federico II

Repository didattico per un laboratorio universitario su **churn prediction** come problema di classificazione, con focus su preparazione del dato, modellazione, valutazione e interpretabilità.

## Lezioni (5)

1. **Il churn come problema di classificazione**: formulazione e analisi esplorativa (EDA)
2. **Dal dato grezzo al dataset modellabile**: pulizia, preprocessing e pipeline
3. **Modelli di classificazione e metriche di valutazione**: confronto modelli e valutazione
4. **Feature importance e interpretabilità**: importanza delle variabili e lettura del modello
5. **Modellazione avanzata e simulazione di progetto**: tuning, validazione e flusso end-to-end

## Ambiente di lavoro: GitHub Codespaces

L'ambiente ufficiale del laboratorio è **GitHub Codespaces**: un ambiente di sviluppo cloud pre-configurato, accessibile direttamente dal browser senza installare nulla in locale.

### Come avviare un Codespace

1. Vai alla pagina principale del repository su GitHub.
2. Clicca sul pulsante verde **`< > Code`**.
3. Seleziona la scheda **Codespaces**.
4. Clicca su **"Create codespace on main"** (o sul branch desiderato).
5. Attendi l'avvio dell'ambiente (circa 1–2 minuti al primo avvio).

Una volta avviato, troverai VS Code nel browser con il repository già clonato, le dipendenze installate e il virtual environment attivo.

> **Nota:** ogni studente ha a disposizione un numero limitato di ore gratuite al mese su GitHub Codespaces. Ricordati di **fermare il Codespace** al termine della sessione tramite il menu `< > Code → Codespaces → Stop codespace`.
