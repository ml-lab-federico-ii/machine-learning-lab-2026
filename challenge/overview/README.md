# Churn Prediction Challenge — Overview

## Obiettivo

Costruire un modello di churn prediction che **massimizzi la ROC-AUC**.

## Valutazione

La valutazione finale avviene su un **test set nascosto** (non disponibile agli studenti).

## Submission

Caricare un file CSV con le colonne:

- `id`
- `churn_probability` (probabilità stimata di churn, tra 0 e 1)
