# Environment

Setup minimale per seguire i notebook del laboratorio.

## Prerequisiti

- Python 3.10+ (consigliato)

## Setup (Windows / PowerShell)

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Avvio notebook

```powershell
jupyter notebook
```
